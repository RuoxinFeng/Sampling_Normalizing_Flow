remove_invalid_sessions.remove_invalid_sessions.remove_invalid_sessions(df: DataFrame, spec: InvalidSessionsSpec) → Tuple[DataFrame, TotalFailureInfo][source]#

Removes invalid sessions (degraded, invalid, total failure) given by json files for the specified SRS and sensors and calculates the posterior total failure probability with a uniform prior and the number of observed total failures.

Parameters:

  • df (pd.DataFrame) – A dataframe containing IFVs and SPVs with their respective Codebeamer IDs.

  • spec (InvalidSessionsSpec) – An object containing specifications about the files that define which sessions need to be removed.

Returns:

  • pd.DataFrame – The dataframe without the sessions that should be removed.

  • TotalFailureInfo – Information about the posterior total failure probability and the observed total failures.

Raises:

  • ValueError – If ‘global’ is part of the sensor_names list.

  • ValueError – If duplicates of session ids exist in the DataFrame.

Notes

This function requires a json file per SRS for every ReasonsToRemove in reasons_to_remove.
